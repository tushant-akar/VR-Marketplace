üìñ Overview

This API provides complete user authentication functionality including registration, login, profile management, and secure session handling. All endpoints return JSON responses with consistent structure.

Base URL

https://your-site.netlify.app/api/

Response Format

{ "success": true|false, "message": "Human readable message", "data": { ... } | null, "error": { ... } | null, "timestamp": "2025-06-24T12:30:00.000Z" }

üîê Authentication

Protected endpoints require a Bearer token in the Authorization header:

Authorization: Bearer YOUR_ACCESS_TOKEN_HERE

Token Types

Token Type Purpose Expiry Usage
Access Token API authentication 1 hour Include in Authorization header
Refresh Token Get new access tokens 7 days Send to /api/auth-refresh

üîó API Endpoints

POST User Registration No Auth Required
/api/auth-register
Register a new user account with email, password, and profile information.

üì• Request Parameters

email (string) REQUIRED

Valid email address for the user account

password (string) REQUIRED

Strong password meeting security requirements

name (string) REQUIRED

User's full name (2-100 characters)

phoneNumber (string) OPTIONAL

User's phone number in international format

dateOfBirth (string) OPTIONAL

Date of birth in YYYY-MM-DD format

profileImageUrl (string) OPTIONAL

URL to user's profile image

üîí Password Requirements:

  • Minimum 8 characters
  • At least 1 uppercase letter (A-Z)
  • At least 1 lowercase letter (a-z)
  • At least 1 number (0-9)
  • At least 1 special character (@$!%*?&)

üì§ Request Example

{ "email": "john.doe@example.com", "password": "SecurePass123!", "name": "John Doe", "phoneNumber": "+1234567890", "dateOfBirth": "1990-05-15", "profileImageUrl": "https://example.com/avatar.jpg" }

Quick Start Example

// Register a new user const response = await fetch('/api/auth/register', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email: 'user@example.com', password: 'SecurePass123!', name: 'John Doe' }) }); const result = await response.json(); console.log(result.data.user); // User object console.log(result.data.token); // Access token
